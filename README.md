# Simulation der Sterbewahrscheinlichkeit und Versicherungszahlungen

## Projektbeschreibung
Dieses Projekt simuliert die Sterbewahrscheinlichkeit einer Gruppe von Versicherungsnehmern innerhalb eines Jahres und berechnet die möglichen Versicherungszahlungen.  
Es demonstriert grundlegende Fähigkeiten in Wahrscheinlichkeitsrechnung, Monte-Carlo-Simulationen und Datenvisualisierung – zentrale Kompetenzen für den Einstieg in die Aktuarwissenschaften.

## Anforderungen
- Python 3.x
- numpy
- matplotlib
- seaborn

## Installation
1. Klonen Sie das Repository:
git clone https://github.com/Zweistein019/versicherungs-simulation.git
2. Installieren Sie die benötigten Bibliotheken:
pip install numpy matplotlib seaborn

## Verwendung
1. Wechseln Sie in das Projektverzeichnis:
cd insurance-simulation
2. Führen Sie das Python-Skript aus:
python simulation.py

## Ergebnis
Das Skript gibt die Gesamtsumme der Versicherungszahlungen und die Anzahl der Todesfälle in der Konsole aus.
Außerdem wird ein Diagramm erstellt und automatisch im Ordner images/ gespeichert.
Beispielausgabe:
Gesamtsumme der Versicherungszahlungen in diesem Jahr: €1,200,000
Anzahl der Todesfälle: 12

## Visualisierung

Das Skript erstellt ein Histogramm der Versicherungszahlungen pro Person.
